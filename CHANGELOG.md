## [v0.1.0] - 2026-06-01 ([`3cb7165`](https://github.com/ProductBuildersHQ/lever/commit/3cb7165))

### Highlights

- LEVER Capability Framework - a five-stage progression model synthesizing classical learning frameworks for the AI era
- Go-first type system with JSON Schema generation and TypeScript/Zod exports
- Mappings to 5 classical frameworks: Bloom's Taxonomy, Dreyfus Model, Shu-Ha-Ri, Apprenticeship, Conscious Competence
