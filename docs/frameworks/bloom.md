# Bloom's Taxonomy

**Origin:** Benjamin Bloom, 1956 (revised 2001)

**Primary Focus:** Cognitive mastery levels

## Overview

Bloom's Taxonomy is a hierarchical model of cognitive learning objectives widely used in education. It describes how learners progress from basic recall to complex creation.

## The Six Levels

| Level | Description | Example |
|-------|-------------|---------|
| **Remember** | Recall facts and basic concepts | Define Kubernetes |
| **Understand** | Explain ideas or concepts | Explain how pods work |
| **Apply** | Use information in new situations | Deploy an application |
| **Analyze** | Draw connections among ideas | Compare deployment strategies |
| **Evaluate** | Justify a decision or course of action | Choose between EKS and GKE |
| **Create** | Produce new or original work | Design a new platform architecture |

## Mapping to LEVER

| LEVER | Bloom |
|-------|-------|
| Learn | Remember, Understand |
| Execute | Apply |
| Value | Analyze, Evaluate |
| Enable | Create |
| Replicate | Beyond Bloom |

### Analysis

**Learn ↔ Remember, Understand**

Both focus on knowledge acquisition. Bloom's Remember and Understand correspond to building foundational knowledge in LEVER's Learn stage.

**Execute ↔ Apply**

Direct correspondence. Both describe using learned knowledge to accomplish tasks.

**Value ↔ Analyze, Evaluate**

Creating independent value requires analysis and judgment—Bloom's higher cognitive levels.

**Enable ↔ Create**

Enabling others often involves creating new approaches, training materials, and systems. Bloom's Create aligns with the creative aspect of enablement.

**Replicate ↔ Beyond Bloom**

Bloom's Taxonomy focuses on individual cognitive mastery. It doesn't address multiplying capability through systems, platforms, or institutions. This is where LEVER extends beyond Bloom.

## Strengths of Bloom

- Well-established in education
- Clear cognitive hierarchy
- Widely understood vocabulary
- Excellent for curriculum design

## Limitations for LEVER's Purpose

- Focuses on individual cognition, not leadership
- Doesn't address multiplication or systems
- No stages for teaching or developing others
- Designed for educational settings, not professional development

## Key Insight

> Bloom describes **what** you can do cognitively. LEVER describes **how** that cognition translates to capability and ultimately to multiplication.
