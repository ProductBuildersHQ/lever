# Execute

**Stage 2 of 5** • Personal Capability

!!! abstract "Core Question"
    **What can I do?**

## Overview

The Execute stage is about applying knowledge to produce work. Practitioners move from theory to practice, completing tasks with guidance and supervision. This is where knowledge becomes skill.

## Output

**Results** — Completed work, delivered tasks, tangible outputs.

## Characteristics

- Completes assigned tasks
- Applies learned techniques
- Requires supervision and guidance
- Produces work product with support

## What "Execute" Looks Like

### Product Builder

- Creates functional prototypes
- Uses AI tools to accelerate development
- Completes features with supervision
- Participates in product delivery

### Software Engineer

- Completes assigned tasks
- Writes code that passes review
- Debugs issues with support
- Contributes to team projects

## Evidence of Execution

- Delivers assigned work on time
- Code/work passes peer review
- Can follow established processes
- Shows improving efficiency over time

## The Supervision Gradient

Execute spans a range of independence:

| Early Execute | Late Execute |
|---------------|--------------|
| Needs detailed instructions | Understands intent |
| Requires frequent check-ins | Self-manages daily work |
| Asks "how" questions | Asks "why" questions |
| Follows patterns exactly | Recognizes when patterns apply |

## Common Pitfalls

!!! warning "Dependency on Instructions"
    Waiting for detailed specifications rather than developing judgment. The goal is to grow toward independence.

!!! warning "Avoiding Feedback"
    Not seeking code reviews or input. Feedback accelerates skill development.

!!! warning "Premature Independence"
    Trying to work alone before having sufficient skill. Execution improves through collaboration.

## Transition to Value

A practitioner is ready to transition to [Value](value.md) when they:

- Can complete work without constant supervision
- Make reasonable decisions independently
- Understand trade-offs in their domain
- Have demonstrated reliable delivery

## Key Insight

> Execution is deliberate and effortful. It requires conscious attention and often feels slow. This is normal—automaticity comes later.

In the AI era, execution can be augmented by AI assistants. A practitioner who learns to leverage AI tools can accelerate their path from Execute to Value, but the fundamental skill development still requires practice.
