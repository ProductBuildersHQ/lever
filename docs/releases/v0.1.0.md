## The Five Stages

LEVER defines a universal capability progression:

| Stage | Question | Output |
|-------|----------|--------|
| **Learn** | Can I follow instructions? | Knowledge |
| **Execute** | Can I complete tasks? | Work |
| **Value** | Can I independently create outcomes? | Outcomes |
| **Enable** | Can I develop others? | People |
| **Replicate** | Can I multiply my impact? | Systems |

The first three stages (L-E-V) represent **personal capability**. The last two (E-R) represent **leveraged capability** — multiplying impact through others and systems.

### Go Library

### CLI

```bash
# Install
go install github.com/ProductBuildersHQ/lever/cmd/lever@latest

# Commands
lever stages          # List LEVER stages
lever spec            # Output complete spec as JSON
lever schema          # Generate JSON Schema
lever profile-schema  # Generate Profile schema
lever mappings        # Output mapping table
lever version         # Print spec version
```

### TypeScript/Zod
